Replace and upload a new list of vehicles to Account

To add vehicles to this account, you can either use your own spreadsheet or download a template (CSV, 1KB).

When filling in the number plates follow these rules so the data uploads with no errors:

  • in cell A1, enter the words 'number plate'
  • the number plates can only be between 2 and 7 characters
  • use only letters and numbers
  • enter number plates in the first column only
  • no empty rows between the number plates
  • the maximum amount of vehicles you can upload is 200,000

Once the file is filled in:

  1. Save it as .CSV - the name of the file does not matter
    (Go to file > save as > .CSV (comma delimited)).
  2. Choose the file.
  3. Upload the file.
